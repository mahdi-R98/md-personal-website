درباره من

من یک توسعه‌دهنده وب هستم با تمرکز بر طراحی رابط‌های کاربری مدرن، سریع و قابل استفاده. مسیر من از علاقه به گرافیک شروع شد و به دنیای کدنویسی ختم شد؛ جایی که می‌تونم خلاقیت و منطق رو با هم ترکیب کنم.

در طول مسیر با ابزارها و فریم‌ورک‌های مختلفی کار کردم، از HTML و CSS گرفته تا Laravel و React. عاشق یادگیریم و همیشه دنبال چالش‌های تازه‌ام. هدفم ساخت تجربه‌های دیجیتال خاص و کاربردی برای کاربران واقعی‌ست.

  • طراحی رابط کاربری با HTML/CSS و Tailwind
  • توسعه سمت سرور با PHP و Laravel
  • کار با پایگاه داده MySQL و SQL
  • ساخت تعاملات پویا با JavaScript و jQuery
  • توسعه ابزارهای اتوماسیون با Python
